## 支持的 ID

### Android (AOSP & Google)

- SSAID (Android ID)
- AAID (广告 ID)

### 小米 (小米 & 红米 & POCO)

- OAID (开放匿名设备标识符)
- VAID (开发者匿名设备标识符)
- Extm ID
- GC ID
- Key ID

### OPPO (OPPO & 一加 & 真我)

- Security ID
- OUID (开放匿名设备标识符)
- GUID (设备唯一标识符)
- 百度 ID

### 华为 (华为 & 荣耀(旧))

- OAID (开放匿名设备标识符)
